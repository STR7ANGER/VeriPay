# Judging Narrative and Submission Copy

## Judge-Facing Positioning

`VeriPay Agent makes AI-driven payments trustworthy. Instead of letting an agent directly control a wallet, we force every payment through intent extraction, policy checks, human approval, and verifiable proof.`

## What Makes This Different

- most AI wallet demos optimize for automation
- we optimize for trust, control, and verification
- the AI is useful, but never blindly trusted
- the human remains in control without losing convenience

## Innovation Angle

We are combining:

- natural-language AI intent parsing
- deterministic policy enforcement
- wallet-based approval
- verifiable execution records

That creates a practical trust layer for agentic commerce.

## Why It Matters

As AI agents become more capable, they will handle financial actions for users and businesses. The missing layer is not just execution, but accountable execution. Our system provides that missing layer.

## Submission Copy Draft

### Project Name

`VeriPay Agent`

### Tagline

`Verifiable payments for AI agents`

### Short Description

VeriPay Agent is a trust layer for AI-driven crypto payments. It converts natural-language payment requests into structured intents, checks them against deterministic safety policies, requires explicit human approval, executes through the user’s wallet, and creates a verifiable audit trail for every payment.

### Problem

AI agents can already trigger actions, but financial actions are still unsafe because users cannot reliably inspect or prove what the agent intended, why a transaction was allowed, or whether policy rules were followed.

### Solution

We built a payment workflow where AI helps interpret the request, but cannot directly move funds. Every payment passes through schema validation, policy checks, approval gates, wallet-based execution, and proof generation.

### Key Features

- natural-language to payment intent
- deterministic policy engine
- explicit human approval
- wallet-based transaction execution
- audit timeline and proof hash

### Stack

- Next.js
- TypeScript
- Tailwind
- Gemini
- Supabase
- viem
- wagmi
- Base Sepolia
- Phantom wallet

### Human-Agent Collaboration Angle

The human defines the payment goal and approves execution. The agent translates intent, explains risk, prepares data, and reduces friction. This shows a real collaborative workflow instead of replacing the human operator.
