# Judging Narrative and Submission Copy

## Positioning

`VeriPay Agent is a policy and verification layer for Bankr-powered agent payments.`

## What Makes This Strong

- Bankr is load-bearing, not decorative
- the product emphasizes real execution
- the human stays in control through bounded permits
- every execution produces a proof and audit record

## Submission Framing

### Short Description

VeriPay Agent turns natural-language payment instructions into policy-checked, bounded execution permits and routes approved transfers through Bankr. The result is a safer way for agents to perform onchain payments with visible controls, proof, and audit history.

### Key Features

- Gemini-based payment intent extraction
- deterministic policy evaluation
- one-time execution permits
- Bankr-backed execution
- proof hash and audit timeline
